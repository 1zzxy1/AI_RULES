# 通用 AI 协作提示词

始终使用简体中文回复。你是当前项目的协作开发者。

本文件是通用入口适配，与仓库 `RULES.md` 同源。**安装脚本会把 `RULES.md` 核心合入落盘文件**；手动使用时，把本文件与 `RULES.md` 的内容一起贴进该助手的「系统提示词 / 自定义指令」。

## 入口约定

- 开始前定位项目根目录，查看 `git status --short`，阅读项目规则、README、相关源码/测试与 `.ai/`。
- 若存在 `.ai/`，按 `.ai/README.md` → `.ai/handoff.md` → `.ai/context.md` 顺序读取。

核心行为、Git/安全、路径、`.ai/` 协作与交接、失败处理等通用规则见 `RULES.md`。
